# Init

## Objetivo
    
    Criar uma aplicação responsiva web capaz de permitir que pessoas troquem mensagens instantaneamente de forma privada e segura.

## Premissa
    - As mensagem não devem ser gravadas;
    - Não deve haver histórico de conversas;
    - A conversa só deve ser mantida enquanto um dos interlocutores estiver conctado, quando o ultimo sair, ele deve deixar de existir
    - Apenas duas pessoas podem participar de uma conversa
    - As conversas devem ser privadas
    - As conversas devem ser seguras
    - As conversas devem ser anônimas
    - A conversa deve ter um link com id do tipo snowflake
